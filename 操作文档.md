## 系统概述

业余球队管理系统是一个用于管理球队、活动和出勤情况的 Web 应用程序。系统提供用户注册、登录、球队管理、活动创建和出勤记录等功能。

### 主要功能

- ✅ 用户注册与登录
- ✅ 球队创建与管理
- ✅ 活动创建与管理
- ✅ 出勤记录与统计
- ✅ 用户信息查看

---

## 访问地址

- **本地开发环境**: `http://localhost:3000`
- **首页**: `http://localhost:3000/` 或 `http://localhost:3000/index.html`
- **登录页面**: `http://localhost:3000/login.html`
- **注册页面**: `http://localhost:3000/register.html`

## 页面功能说明

### 首页

**访问路径**: `/` 或 `/index.html`

**功能说明**:
- 系统欢迎页面，展示系统状态
- 提供快速访问登录和注册功能的入口
- 显示系统健康状态检查

**操作步骤**:
1. 打开浏览器访问 `http://localhost:3000`
2. 页面会自动检查系统状态
3. 点击"登录"按钮跳转到登录页面
4. 点击"注册"按钮跳转到注册页面
5. 可以测试 API 接口（注册、登录、获取用户信息）

**注意事项**:
- 系统状态每 30 秒自动更新一次
- 需要后端服务正常运行才能显示正确的系统状态

### 用户注册

**访问路径**: `/register.html`

**功能说明**:
- 新用户注册账户
- 创建用户账号后自动跳转到登录页面

**操作步骤**:
1. 访问注册页面或从首页点击"注册"按钮
2. 填写注册信息：
   - **邮箱**: 输入有效的邮箱地址（必填）
   - **用户名**: 输入用户名（必填）
   - **密码**: 输入密码，至少 6 位（必填）
3. 点击"注册"按钮提交
4. 注册成功后，页面会显示成功提示，并在 1.5 秒后自动跳转到登录页面

**字段说明**:
- **邮箱**: 用于登录和接收通知，必须是有效的邮箱格式
- **用户名**: 用于显示和登录，建议使用易记的名称
- **密码**: 至少 6 个字符，建议使用强密码

**错误处理**:
- 如果邮箱已存在，会显示错误提示
- 如果用户名已存在，会显示错误提示
- 如果网络错误，会显示"网络错误，请稍后重试"

**相关链接**:
- 已有账号？点击"立即登录"跳转到登录页面
- 点击"返回首页"返回系统首页

### 用户登录

**访问路径**: `/login.html`

**功能说明**:
- 用户登录系统
- 登录成功后跳转到用户仪表板

**操作步骤**:
1. 访问登录页面或从首页点击"登录"按钮
2. 填写登录信息：
   - **用户名或邮箱**: 输入注册时的用户名或邮箱（必填）
   - **密码**: 输入注册时的密码（必填）
3. 点击"登录"按钮提交
4. 登录成功后，自动跳转到用户仪表板页面

**登录方式**:
- 可以使用用户名登录
- 可以使用邮箱登录
- 两种方式任选其一即可

**错误处理**:
- 如果用户名/邮箱或密码错误，会显示错误提示
- 如果网络错误，会显示"网络错误，请稍后重试"
- 如果未登录访问需要权限的页面，会自动跳转到登录页面

**相关链接**:
- 还没有账号？点击"立即注册"跳转到注册页面
- 点击"返回首页"返回系统首页

**注意事项**:
- 登录成功后，系统会保存登录令牌（Token）
- Token 保存在浏览器本地存储中，关闭浏览器后需要重新登录
- 如果 Token 过期，需要重新登录

### 用户仪表板

**访问路径**: `/dashboard.html`

**功能说明**:
- 显示用户个人信息
- 显示统计数据（球队数量、活动数量、出勤率等）
- 提供快速操作入口

**访问权限**: 需要登录后才能访问

**页面组成**:

1. **导航栏**
   - 系统标题
   - 导航菜单：仪表板、球队管理、活动管理、出勤管理
   - 用户名显示
   - 退出登录按钮

2. **统计卡片**
   - **我的球队**: 显示用户创建的球队数量
   - **活动数量**: 显示用户创建的活动数量
   - **出勤率**: 显示用户的出勤率百分比
   - **本月出勤**: 显示本月的出勤次数

3. **个人信息卡片**
   - **用户名**: 显示当前登录用户的用户名
   - **邮箱**: 显示当前登录用户的邮箱
   - **注册时间**: 显示账户创建时间
   - **用户ID**: 显示用户的唯一标识符

4. **快速操作**
   - **管理球队**: 跳转到球队管理页面
   - **创建活动**: 跳转到活动管理页面
   - **查看出勤**: 跳转到出勤管理页面

**操作步骤**:
1. 登录后自动跳转到仪表板
2. 查看个人信息和统计数据
3. 点击快速操作按钮跳转到相应功能页面
4. 点击导航栏中的菜单项切换页面
5. 点击"退出登录"按钮退出系统

**注意事项**:
- 如果未登录访问此页面，会自动跳转到登录页面
- 统计数据需要后端 API 支持，如果后端未实现相关功能，可能显示为 0

### 球队管理

**访问路径**: `/teams.html`

**功能说明**:
- 创建新球队
- 查看球队列表
- 编辑球队信息
- 删除球队

**访问权限**: 需要登录后才能访问

**操作步骤**:

#### 创建球队
1. 点击页面右上角的"+ 创建球队"按钮
2. 在弹出的模态框中填写：
   - **球队名称**: 输入球队名称（必填）
   - **球队描述**: 输入球队描述（可选）
3. 点击"保存"按钮
4. 创建成功后，模态框自动关闭，球队列表自动刷新

#### 查看球队列表
- 页面加载时自动显示所有球队
- 每个球队卡片显示：
  - 球队名称
  - 球队描述
  - 成员数量
  - 活动数量
  - 创建时间

#### 编辑球队
1. 在球队列表中，找到要编辑的球队
2. 点击该球队卡片上的"编辑"按钮
3. 在弹出的模态框中修改球队信息
4. 点击"保存"按钮保存修改

#### 删除球队
1. 在球队列表中，找到要删除的球队
2. 点击该球队卡片上的"删除"按钮
3. 在弹出的确认对话框中点击"确定"
4. 删除成功后，球队列表自动刷新

**错误处理**:
- 如果创建/编辑失败，会在表单下方显示红色错误提示
- 如果网络错误，会提示"无法连接到后端服务，请确保后端已启动"
- 如果未登录，会自动跳转到登录页面

**注意事项**:
- 删除操作不可恢复，请谨慎操作
- 球队名称不能为空
- 需要后端服务正常运行才能使用完整功能

### 活动管理

**访问路径**: `/activities.html`

**功能说明**:
- 创建新活动
- 查看活动列表
- 编辑活动信息
- 删除活动

**访问权限**: 需要登录后才能访问

**操作步骤**:

#### 创建活动
1. 点击页面右上角的"+ 创建活动"按钮
2. 在弹出的模态框中填写：
   - **活动名称**: 输入活动名称（必填）
   - **活动日期**: 选择活动日期（必填）
   - **活动时间**: 选择活动时间（必填）
   - **活动地点**: 输入活动地点（必填）
   - **活动描述**: 输入活动描述（可选）
3. 点击"保存"按钮
4. 创建成功后，模态框自动关闭，活动列表自动刷新

#### 查看活动列表
- 页面加载时自动显示所有活动
- 每个活动卡片显示活动的基本信息

#### 编辑活动
1. 在活动列表中，找到要编辑的活动
2. 点击该活动卡片上的"编辑"按钮
3. 在弹出的模态框中修改活动信息
4. 点击"保存"按钮保存修改

#### 删除活动
1. 在活动列表中，找到要删除的活动
2. 点击该活动卡片上的"删除"按钮
3. 在弹出的确认对话框中点击"确定"
4. 删除成功后，活动列表自动刷新

**字段说明**:
- **活动名称**: 活动的名称，如"训练"、"比赛"等
- **活动日期**: 活动举行的日期
- **活动时间**: 活动开始的时间
- **活动地点**: 活动举行的地点
- **活动描述**: 活动的详细说明

**注意事项**:
- 所有必填字段必须填写
- 活动日期和时间需要选择有效的日期和时间
- 需要后端服务正常运行才能使用完整功能

### 出勤管理

**访问路径**: `/attendance.html`

**功能说明**:
- 查看出勤记录
- 筛选出勤记录（按活动、日期范围）
- 查看出勤统计信息

**访问权限**: 需要登录后才能访问

**页面组成**:

1. **筛选区域**
   - **选择活动**: 下拉菜单选择特定活动，或选择"全部活动"
   - **选择日期范围**: 选择开始日期和结束日期

2. **统计卡片**
   - **总出勤次数**: 显示总的出勤次数
   - **出勤率**: 显示出勤率百分比
   - **缺席次数**: 显示缺席次数

3. **出勤记录列表**
   - 显示所有出勤记录的详细信息

**操作步骤**:

#### 查看出勤记录
1. 页面加载时自动显示所有出勤记录
2. 记录列表显示：
   - 活动名称
   - 出勤状态（出席/缺席/迟到）
   - 出勤日期
   - 备注信息

#### 筛选出勤记录
1. **按活动筛选**:
   - 在"选择活动"下拉菜单中选择特定活动
   - 选择"全部活动"显示所有记录

2. **按日期范围筛选**:
   - 在"选择日期范围"中选择开始日期
   - 选择结束日期
   - 系统自动筛选该日期范围内的记录

#### 查看统计信息
- 页面顶部的统计卡片自动显示：
  - 总出勤次数
  - 出勤率
  - 缺席次数

**出勤状态说明**:
- **present**: 出席
- **absent**: 缺席
- **late**: 迟到

**注意事项**:
- 筛选条件可以组合使用
- 统计数据会根据筛选条件自动更新
- 需要后端服务正常运行才能使用完整功能

## 常见问题

### 1. 无法访问页面

**问题**: 打开页面显示"无法访问此网站"或空白页面

**解决方案**:
- 确保后端服务已启动（运行 `npm run dev`）
- 检查浏览器地址栏的 URL 是否正确
- 确认端口号是否为 3000

### 2. 登录后自动跳转到登录页面

**问题**: 登录成功后，访问其他页面时又跳转回登录页面

**解决方案**:
- 检查浏览器是否禁用了本地存储（localStorage）
- 清除浏览器缓存后重新登录
- 检查后端服务是否正常运行

### 3. 创建球队/活动失败

**问题**: 点击保存按钮后显示错误提示

**解决方案**:
- 确保后端服务正在运行
- 检查网络连接
- 查看浏览器控制台的错误信息
- 确认已正确登录

### 4. 页面显示"加载中"但一直不更新

**问题**: 页面显示"加载中..."但数据一直不显示

**解决方案**:
- 检查后端 API 是否正常响应
- 打开浏览器开发者工具（F12）查看网络请求
- 确认后端服务已启动且数据库连接正常

### 5. 统计数据显示为 0

**问题**: 仪表板上的统计数据都显示为 0

**解决方案**:
- 这是正常的，如果还没有创建球队或活动
- 如果已创建数据但仍显示 0，检查后端 API 是否实现了统计功能
- 刷新页面重新加载数据

### 6. 忘记密码

**问题**: 忘记登录密码

**解决方案**:
- 目前系统不支持密码重置功能
- 需要联系管理员或重新注册账户
- 建议使用密码管理器保存密码

### 7. 浏览器兼容性

**问题**: 某些功能在特定浏览器中无法使用

**解决方案**:
- 推荐使用现代浏览器（Chrome、Firefox、Safari、Edge）
- 确保浏览器版本较新
- 启用 JavaScript 功能
